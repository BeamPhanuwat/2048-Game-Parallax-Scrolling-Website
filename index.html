2048 GAME คืออะไร?

2048 เป็นเกมปริศนาเคลื่อนบล็อก (sliding block puzzle) แบบเล่นคนเดียว ซึ่งออกแบบโดยนักพัฒนาเว็บชาวอิตาลี Gabriele Cirulli เป้าหมายของเกมคือเพื่อให้ผู้เล่นเลื่อนกระเบื้องตัวเลขบนกริดไปรวมกันเพื่อให้เกิดจำนวน 2048 ขึ้น อย่างไรก็ตามผู้เล่นสามารถเล่นต่อได้ถึงแม้จะถึงเป้าหมายเกมที่เลข 2048 แล้วก็ตามเพื่อสร้างกระเบื้องตัวเลขที่จำนวนสูงขึ้นไปอีก

2048 ดั้งเดิมแล้วเขียนโดยใช้จาวาสคริปต์ และ CSS ระหว่างช่วงสุดสัปดาห์ของ Gabriele Cirulli ก่อนที่จะเปิดตัวเมื่อวันที่ 9 มีนาคม 2014 ในรูปแบบซอฟต์แวร์ไม่เสียค่าใช้จ่ายและโอเพนซอร์ซ (free and open-source software) ตาม สัญญาอนุญาตของเอ็มไอที เกมแบบเดียวกันที่เขียนโดยใช้ C++ และ Vala ถูกปล่อยออกมาตามกันในภายหลัง

2048 นั้นถูกอธิบายว่าคล้ายคลึงกับแอปเกม Threes! ซึ่งปล่อยออกมาเมื่อเดือนก่อนหน้าแต่ Cirulli อธิบายด้วยตนเองว่า 2048 เป็นวิดีโอเกมโคลน (Video game clone) ของแอปเกม 1024 ที่ผลิตโดย Veewo Studios
